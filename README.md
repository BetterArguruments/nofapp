# nofapp
Awesomeness.

## Build
If you haven't installed Cordova and Ionic yet, do it:
```
sudo npm install -g cordova ionic
```
Clone this
```
git clone
```
Install Node.js and Bower Dependencies
```
npm install
bower install
```
